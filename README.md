# Eduleb

Eduleb is a Django-based educational platform built to practice and develop real-world backend development skills.

## Overview

This project focuses on building a web application using Python and Django, including backend logic, database management, and web application features.

## Features

- User authentication
- Django-based backend structure
- Database management
- Web application functionality

## Technologies

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

## Installation

Clone the repository:

```bash
git clone https://github.com/CsGho3T/eduleb-django.git

Navigate to the project directory:

cd eduleb-django

Create a virtual environment:

python -m venv venv

Install dependencies:

pip install -r requirements.txt

Run migrations:

python manage.py migrate

Start the development server:

python manage.py runserver
