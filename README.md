# Eduleb

Eduleb is a Django-based educational platform built to practice and develop real-world backend development skills.

## Overview

This project focuses on building a web application using Python and Django, including backend logic, database management, and web application features.

## Features

- User authentication
- Django-based backend structure
- Database management
- Web application functionality

## Technologies

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap
